# SIG名称
FangTian是一款围绕视窗显示、视窗管理，持续构建关键技术的视窗引擎，依托视窗的创新架构，实现多生态的高质量兼容，为用户提供高性能、低能耗的流畅操控体验，打造一个完整的生态繁荣的视窗引擎
